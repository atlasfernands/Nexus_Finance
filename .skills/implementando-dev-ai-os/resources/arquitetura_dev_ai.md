Recursos para implementação do Dev AI OS:

- **Arquitetura**: VS Code Extension + Local LLM API + RAG + Tools + Dashboard
- **Tecnologias**: Ollama/vLLM, TypeScript, Electron, ChromaDB, LangChain
- **Modelos**: LLaMA 70B para geração, embeddings para RAG
- **Integração**: HTTP APIs, WebSocket para comunicação em tempo real
- **Segurança**: Execução local, sem dados enviados para nuvem

Use estes recursos como base para customizar e expandir o sistema.
